# Simple Notes App (Flask)

A minimal web app built with Python Flask. 
Users can add notes, view them, and delete them.

## Features
- Add a note
- View all notes
- Delete a note
- Clean responsive UI with HTML + CSS

## Tech
- Python (Flask)
- HTML, CSS

## How to run
1. Install Flask:
   pip install flask
2. Run the app:
   python app.py
3. Open in browser:
   http://127.0.0.1:5000/
